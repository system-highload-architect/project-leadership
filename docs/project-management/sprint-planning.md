# План спринта: RMS (Robot Management System)

**Спринт №:** 1 (MVP-прототип)  
**Длительность:** 3 недели (03.07.2026 – 24.07.2026)  
**Цель спринта:** Создать работоспособное ядро системы (базовые сущности, CRUD, события, CI/CD).

---

## 1. Цель спринта

Разработать минимально жизнеспособный продукт (MVP), позволяющий:
- Создавать, редактировать, удалять роботов, задачи, заказы, локации через REST API.
- Публиковать события о всех изменениях в Kafka.
- Развернуть систему в dev-окружении (Kubernetes).
- Написать базовые интеграционные тесты и документацию API.

## 7. Критерии завершения спринта

- Все задачи S-01…S-08 выполнены и протестированы.
- Система развёрнута в dev-окружении.
- Базовый функционал работает через REST API.
- События публикуются в Kafka.
- Интеграционные тесты проходят успешно.
- Документация OpenAPI актуальна.
